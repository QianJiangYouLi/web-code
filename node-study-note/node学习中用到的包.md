# node-stydy-note

node学习记录

# 用到的第三方模块

## 01 moment

功能：日期处理

~~~bash
npm i moment@2.29.1
~~~

## 02 nodemon

功能：node代码热更新

~~~bash
npm i -g nodemon

#  mac 本
sudo npm i -g nodemon
~~~

## 03 nrm

功能：换源

~~~bash
npm i -g nrm

#  mac 本
sudo npm i -g nrm
~~~

## 04 then-fs

处理fs模块的回调地狱问题。

~~~bash
npm install then-fs
~~~

## 05 express

快速搭建服务器

~~~bash
npm i express@4.17.1
~~~

## 06 multer
~~~bash
npm i multer@1.4.2
~~~

## 07 cors
处理跨域
~~~bash
npm i cors@2.8.5
~~~
